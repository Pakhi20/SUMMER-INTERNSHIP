# Automated Recognition and Analysis of Bharatanatyam Adavus

## Project Overview
This project focuses on the automated recognition and analysis of Bharatanatyam Adavus, which are the fundamental steps of the classical Indian dance form Bharatanatyam. The system processes annotated dance video data to extract key postures and transitional frames for analysis.

## Objective
The objective of this project is to develop a system that can organize and analyze Bharatanatyam dance movements by identifying important postures and motion sequences from annotated datasets.

## Features
- Extraction of key posture frames from annotated dance videos
- Structured storage of posture data in a database
- Visualization of extracted frames using a GUI
- Organized dataset for Bharatanatyam Adavus analysis

## Technologies Used
- Java
- Python
- SQLite
- Pandas
- NumPy
- Java Swing
- Google Colab
- Visual Studio Code

## Applications
This system can help dancers, researchers, and students analyze Bharatanatyam movements, study posture patterns, and support the development of AI-based dance learning tools.

## Future Scope
- Automated Adavu recognition using computer vision
- Real-time posture detection and feedback
- Extension to other classical Indian dance forms
